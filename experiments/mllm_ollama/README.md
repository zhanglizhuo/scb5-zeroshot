# MLLM Ollama Evaluation Scripts (Legacy)

These scripts orchestrate MLLM evaluations via [Ollama](https://ollama.ai) for
Qwen 2.5-27B, Gemma 4-26B, and Qwen 3. They are preserved as reference for the
MLLM evaluation methodology used in the paper.

**Note:** Internal paths reference the old directory structure and may need
updating before use. The canonical MLLM entry point is `experiments/main_mllm.py`.
